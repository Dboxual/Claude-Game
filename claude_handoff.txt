What feels unfinished
---------------------
- No real multiplayer yet; the server browser is a UI shell and Direct Connect does not connect.
- Dedicated server does not yet load server/content/ or simulate combat/world gameplay.
- Combat tuning constants live in code, mostly src/shared/melee.h, not data files.
- Sound-event mapping is still client-side code.
- No player health/death loop; taking hits currently produces feedback instead of a full gameplay consequence.
- Duelist Bot is only a training partner. It does not move, kick, feint, parry, or counter.
- Visual validation of the latest melee poses/animations was not completed by Claude. First manual playtest should check shield raise, guard pose, kick boot, bot telegraphs, and thrown weapon orientation.

What needs tuning next
----------------------
- Manual feel pass on melee timing, stamina costs, parry/perfect-counter windows, guard break length, and bot windup readability.
- Review interaction edge cases around gear pickups, carried props, thrown weapons landing as pickups, respawn timing, and save/load.
- Consider moving stable combat balance values from melee.h into data/content once the feel settles.
- Wire server-side content loading and shared simulation when networking/server authority work begins.
- Consider extracting growing client combat orchestration from Game into a smaller controller before it gets harder to review.

How to run the client
---------------------
One-click Windows path:
  run_windows.bat

After building manually:
  build\Release\tacmove.exe

Useful smoke/dev flags:
  build\Release\tacmove.exe --world --frames 240 --novsync
  build\Release\tacmove.exe --world --equip sword

Manual combat smoke from Claude's note:
  1. run_windows.bat, then Singleplayer -> Start Test World, or start with --equip sword.
  2. B -> Bots -> DUELIST BOT, then B again to close the menu.
  3. Parry: tap RMB just before the bot strike after the cue/amber blade, then attack during the riposte window.
  4. Perfect counter: match the incoming attack type as it lands, for example wheel-up against overhead.
  5. Kick: when the bot guards, walk in and press F to open the guard.
  6. Shield: B -> Pickups -> SHIELD, E to grab, hold RMB to block.
  7. Throw: with sword or karambit held, press Q; E picks it back up after it lands.
